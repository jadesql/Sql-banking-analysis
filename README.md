# Mobile Banking SQL Analysis

## 📊 Project Overview
This project simulates a simple mobile banking system and uses SQL to analyze user transactions.

## 🗂 Dataset
- users: user information (user_id, age)
- transactions: banking activity (deposits and withdrawals)

## ❓ Key Questions Answered
- How many transactions are there?
- What is the total money moved?
- Do users deposit or withdraw more?
- Which users are most active?

## 🔍 Key Insights
- Transaction activity varies by user
- Deposits and withdrawals can be compared using aggregation
- SQL GROUP BY is used for analysis

## 🛠 Tools
- SQLite
- Visual Studio Code
